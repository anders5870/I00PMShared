Name: istrslen
Type: (char s, size_t length)-> char
Post: Sets the length of an istring and inserts a corresponding '\0' character. If the length argument is out of bounds for the string, a new istring should be returned with the contents of the original string. The last character of the original string will be repeated to fill the string to its given length.
