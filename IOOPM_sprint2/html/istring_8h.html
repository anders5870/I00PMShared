size_t istrfixlen ( char *  s)

Name: istrfixlen
Type: char *s->size_t
Post: Inspects the length of the string from str and if it differs from length, updates length accordingly. If str does not contain a \0 token, it will be inserted at the first element which is not a printable ascii character, and length updated accordingly. This function must be as efficient as possible and not rescan the string if it can be avoided.
This function is useful when istrings have been manipulated as regular C strings, to reestablish the length invariant.
